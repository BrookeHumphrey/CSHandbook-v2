Provider Setup & Section Mapping

This page details the process for setting up new providers, mapping sections in Athena, and verifying workflow for AI and SDK integration.

New Provider Setup

  1. Check if the provider already exists in Athena using the Practice ID and username.
  2. If not, create a new provider account with appropriate attributes:
FieldEntry
iScribe UsernameAthena Username + Clinic ID (e.g., p-mcardwell20)
PartnerAthena SDK
Display NameFull name as listed in Athena
EMRAthena
DeviceiPad, iPhone, or PC

Section Mapping

To ensure AI notes are saved correctly in Athena:

  • Verify section mapping for AI Summary and custom macros.
  • Check flags for each section to ensure proper workflow.
  • Confirm that new workflows are reflected in iScribe before testing.

Workflow Verification

  • Test a sample dictation to ensure notes are created correctly.
  • Review AI Summary confidence scores and Correlation IDs.
  • Escalate any errors according to the SDK/AI ticketing process.

Troubleshooting Steps

  • Validate that provider and clinic IDs match.
  • Check the NPS and mapping configurations.
  • Review device connectivity and workflow.
  • Escalate unresolved issues following the Level 3 support protocol.
